# (C# 코딩) Simple Calculator

## 개요
- C# 프로그래밍 학습
- 1줄 소개: C#으로 구현한 간단한 계산기
- 사용한 플랫폼:
 - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
 - TextBox, Button
- 사용한 기술과 구현한 기능
 - Visual Studio를 이용하여 UI 디자인
 - 사칙연산 시 연산자만 변경하고 동일 로직으로 계산 수행
 - 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)
 - 실수 계산 지원
 - 연산자 우선순위 처리
 - 에러 처리(0으로 나누기 등)
 - log 함수 구현
