# (C# 코딩) Simple Calculator

## 개요
- C# 프로그래밍 학습
- 1줄 소개: C#으로 구현한 간단한 계산기
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - TextBox, Button, Label
- 사용한 기술과 구현한 기능
  - Visual Studio를 이용하여 UI 디자인
  - 사칙연산 시 연산자만 변경하고 동일 로직으로 계산 수행
  - 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)
  - 실수 계산 지원
  - 연산자 우선순위 처리
  - 에러 처리(0으로 나누기 등)
  - log 함수 구현
  - 배경 이미지 구현

- 과제 내용
  -TextBox(입력표시, 결과표시), Button(계산) 등을 적절히 배치합니다.
  -숫자 Button 클릭 시 TextBox에 표시합니다. 2가지방법으로표시
  -2개의 피연산자의 입력값을 Int로 바꾸어 더하기 계산을 수행하고 그 결과를 저장합니다.
  -계산 결과값을 문자열로 변환하여 표시합니다.

- 구현 내용과 기능 설명
  -Visual Studio의 Windows Forms 디자이너를 사용하여 TextBox와 Button을 배치하였습니다.
  -숫자 Button 클릭 시 해당 숫자가 TextBox에 표시되도록 이벤트 핸들러를 구현하였습니다.
  -2개의 피연산자를 Int로 변환하여 덧셈 계산을 수행하는 로직을 작성하였습니다.
  -계산 결과값을 문자열로 변환하여 TextBox에 표시하도록 구현하였습니다.

![과제3 실행 화면](img/계산기3-4.png)

- 과제 내용
  -다음의사례로설명 12 + 100 = 112
  -C 버튼▶현재의 모든 내용을 삭제하고 처음(초기화된) 상태로 되돌아감
  -CE 버튼▶마지막입력한 피연산자(Operand) 값을 삭제함 ▶100 입력 후에 Del 눌렀다면 100값이 통째로 삭제됨
  -Del 버튼▶마지막입력된 글자 하나(숫자 하나)값을 삭제함 ▶100 입력 후에 Del 눌렀다면 10으로 변경됨

- 구현 내용과 기능 설명
  -C 버튼을 클릭하면 현재의 모든 내용을 삭제하고 초기화된 상태로 되돌아가도록 구현하였습니다.
  -CE 버튼을 클릭하면 마지막 입력한 피연산자 값을 삭제하도록 구현하였습니다. 예를 들어, 
  -Del 버튼을 클릭하면 마지막 입력된 글자 하나(숫자 하나)값을 삭제하도록 구현하였습니다.
